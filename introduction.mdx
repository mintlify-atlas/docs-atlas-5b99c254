# Welcome to Mikunki Shapshico

Mikunki Shapshico is a modern festival website celebrating 12 years of gastronomic and cultural tradition from UNHEVAL in Huánuco, Peru. This frontend project features a stunning single-page design with live streaming capabilities, interactive chat, countdown timers, and immersive visual effects.

## What Makes This Special

The Mikunki Shapshico website is built with vanilla HTML, CSS, and JavaScript - no frameworks required. It showcases modern web development techniques while maintaining exceptional performance.

## Core Components

### Navigation
- Fixed header with blur backdrop
- Mobile hamburger menu with smooth transitions
- Active section highlighting based on scroll position
- Accessibility-compliant with ARIA attributes

### Hero Section
- Full viewport height with centered content
- Animated logo with pulse effect
- Real-time countdown to festival date (Nov 18, 2025)
- Call-to-action buttons with hover effects
- Fire video background with blend modes

### Live Streaming Section
- 16:9 aspect ratio player shell
- Vimeo iframe integration (source/script.js:11)
- Waiting screen with festival branding
- Viewer counter with random fluctuation
- Mobile-responsive layout sync

### Gallery
- Masonry grid layout
- Paper frame effect with rotation
- Lightbox modal for full-size viewing
- Lazy loading for performance

<Warning>
  The live stream URL is configured in `script.js` at line 11. Update the `STREAM_URL` constant to point to your Vimeo video.
</Warning>

## Performance Features

- **Lazy Loading**: Images load only when needed
- **Preconnect**: DNS prefetching for Google Fonts
- **RequestAnimationFrame**: Smooth animations without jank
- **Intersection Observer**: Efficient scroll-based reveals
- **LocalStorage**: Persistent user preferences and chat sessions

## Browser Support

The site uses modern web APIs and is optimized for:
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

<Note>
  Features like Intersection Observer and CSS Grid are used extensively. Older browsers may require polyfills.
</Note>
