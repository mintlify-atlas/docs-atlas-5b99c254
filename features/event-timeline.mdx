# Event Timeline

A vertical timeline component that displays the festival schedule with color-coded events, time badges, and interactive hover effects.

## Overview

The event timeline features:
- **Vertical gradient line** connecting events
- **Color-coded dots** for different event types
- **Time badges** with clock icons
- **Category icons** for event classification
- **Hover animations** with sliding and scaling effects
- **Responsive layout** for mobile and desktop

<ParamField path="data-color" type="string" required>

  - `orange` - Orange border (fire-1 color)

    ```html
      </div>
    </article>
    ```
  </Step>
  
  <Step title="Choose Color Categories">
    Assign `data-color` based on event type:
    - `white` - General/Opening events
    - `orange` - Main activities
    - `gold` - Cultural performances
    - `red` - Closing/Awards
  </Step>
  
  <Step title="Select Icons">
    Use Font Awesome icons that match event type:
    - `fa-microphone` - Opening ceremony
    - `fa-bowl-food` - Food competition
    - `fa-music` - Cultural performance
    - `fa-trophy` - Awards
  </Step>
  
  <Step title="No JavaScript Required">
    The timeline is pure CSS - no initialization needed.
    All animations are CSS transitions.
  </Step>
</Steps>

## Integration with Reveal Animations

The timeline works with the `initRevealAnimations()` function:

```javascript
// In script.js:121
function initRevealAnimations() {
  const revealEls = $$('.reveal');
  
  const revealObserver = new IntersectionObserver(
    (entries) => {
      entries.forEach((entry) => {
        if (!entry.isIntersecting) return;
        
        entry.target.classList.add('visible');
        
        // Stagger effect on .event-card children
        const staggerTargets = entry.target.querySelectorAll('.event-card');
        staggerTargets.forEach((child, i) => {
          child.style.opacity = '0';
          child.style.transform = 'translateY(20px)';
          child.style.transition = `opacity 0.5s ease ${i * 0.08}s, transform 0.5s ease ${i * 0.08}s`;
          
          requestAnimationFrame(() => {
            child.style.opacity = '1';
            child.style.transform = 'translateY(0)';
          });
        });
        
        revealObserver.unobserve(entry.target);
      });
    },
    { threshold: 0.15, rootMargin: '0px 0px -50px 0px' }
  );
  
  revealEls.forEach((el) => revealObserver.observe(el));
}
```

**To enable:** Add `.reveal` class to timeline container:

```html
<div class="container reveal">
  <div class="timeline">
    <!-- events -->
  </div>
</div>
```

**Effect:** Each event card fades in and slides up with 80ms stagger delay
