# Countdown Timer

A real-time countdown timer that displays days, hours, minutes, and seconds until the festival begins.

## Overview

The countdown timer:
- **Updates every second** with precise calculations
- **Auto-stops** when the festival date arrives
- **Responsive design** with horizontal cell layout
- **Animated separators** with blinking colons
- **Integrated with CONFIG** for centralized date management

## Configuration

<ParamField path="FESTIVAL_DATE" type="Date" required>
  The target date and time for the countdown.
  
  ```javascript
  const CONFIG = {
    FESTIVAL_DATE: new Date("2025-11-18T10:00:00")
  };
  ```
  
  **Format:** ISO 8601 string (YYYY-MM-DDTHH:mm:ss)
  
  **Timezone:** Uses the user's local timezone
</ParamField>

## Core Function

### initCountdown()

Initializes and manages the countdown timer.

**Location:** `script.js:161`

```javascript
function initCountdown() {
  const daysEl = $("#days");
  const hoursEl = $("#hours");
  const minutesEl = $("#minutes");
  const secondsEl = $("#seconds");

  if (!daysEl || !hoursEl || !minutesEl || !secondsEl) return;

  function updateCounter() {
    const now = Date.now();
    const diff = Math.max(0, CONFIG.FESTIVAL_DATE.getTime() - now);
    
    const d = Math.floor(diff / (1000 * 60 * 60 * 24));
    const h = Math.floor((diff / (1000 * 60 * 60)) % 24);
    const m = Math.floor((diff / (1000 * 60)) % 60);
    const s = Math.floor((diff / 1000) % 60);

    daysEl.textContent = String(d).padStart(2, "0");
    hoursEl.textContent = String(h).padStart(2, "0");
    minutesEl.textContent = String(m).padStart(2, "0");
    secondsEl.textContent = String(s).padStart(2, "0");
  }

  // Initial update
  updateCounter();
  
  // Set interval
  const interval = setInterval(() => {
    const now = Date.now();
    const diff = CONFIG.FESTIVAL_DATE.getTime() - now;
    
    if (diff <= 0) {
      clearInterval(interval);
      daysEl.textContent = "00";
      hoursEl.textContent = "00";
      minutesEl.textContent = "00";
      secondsEl.textContent = "00";
      return;
    }
    
    updateCounter();
  }, 1000);
}
```

### Calculation Logic

**Time Difference:**
```javascript
const diff = Math.max(0, CONFIG.FESTIVAL_DATE.getTime() - Date.now());
```
- `Math.max(0, ...)` prevents negative values
- `getTime()` returns milliseconds since epoch

**Component Extraction:**
```javascript
const d = Math.floor(diff / (1000 * 60 * 60 * 24));  // Days
const h = Math.floor((diff / (1000 * 60 * 60)) % 24); // Hours (0-23)
const m = Math.floor((diff / (1000 * 60)) % 60);      // Minutes (0-59)
const s = Math.floor((diff / 1000) % 60);             // Seconds (0-59)
```

**Formatting:**
```javascript
String(d).padStart(2, "0") // "7" → "07"
```

### Auto-Stop Behavior

When the countdown reaches zero:

```javascript
if (diff <= 0) {
  clearInterval(interval);  // Stop the interval
  daysEl.textContent = "00";
  hoursEl.textContent = "00";
  minutesEl.textContent = "00";
  secondsEl.textContent = "00";
  return;
}
```

## Advanced Customization

### Add Animation on Update

```javascript
function updateCounter() {
  const now = Date.now();
  const diff = Math.max(0, CONFIG.FESTIVAL_DATE.getTime() - now);
  
  const s = Math.floor((diff / 1000) % 60);
  const prevSeconds = secondsEl.textContent;
  
  secondsEl.textContent = String(s).padStart(2, "0");
  
  // Pulse animation on second change
  if (prevSeconds !== secondsEl.textContent) {
    secondsEl.style.transform = 'scale(1.1)';
    setTimeout(() => {
      secondsEl.style.transform = 'scale(1)';
    }, 100);
  }
}
```

**Interval Efficiency:**
- Updates only once per second (not every frame)
- Uses `Date.now()` for accurate time calculation

**Browser Optimization:**
- Modern browsers optimize `setInterval` when tab is inactive
- Minimal DOM manipulation (only 4 text updates per second)

## Troubleshooting

**Countdown shows "00:00:00:00" immediately:**
- Check that `FESTIVAL_DATE` is in the future
- Verify date format: `new Date("YYYY-MM-DDTHH:mm:ss")`

**Numbers don't update:**
- Ensure HTML elements have correct IDs: `#days`, `#hours`, `#minutes`, `#seconds`
- Check that `initCountdown()` is called after DOM loads

**Wrong timezone:**
- `new Date()` uses local timezone by default
- For specific timezone, use libraries like `date-fns-tz` or convert manually
