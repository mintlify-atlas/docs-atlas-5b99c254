## Overview

The Mikunki Shapshico website stores all content directly in `index.html`. This guide shows you how to update each section of the site with your festival's information.

## Hero Section

The hero section is the first thing visitors see. Update the main branding and call-to-action:

<Steps>
  <Step title="Update festival name">
    ```html index.html (lines 58-62)
    <a class="brand" href="#hero">
      <img class="brand-logo" src="YOUR_LOGO_URL" alt="Your Festival Icon" />
      <img class="brand-wordmark" src="YOUR_WORDMARK_URL" alt="Your Festival Name" />
    </a>
    ```
  </Step>
  
  <Step title="Update subtitle and location">
    ```html index.html (line 88)
    <p class="hero-sub">Your Festival Type · Your Organization · Your Location</p>
    ```
  </Step>
  
  <Step title="Update badges">
    ```html index.html (lines 90-93)
    <div class="hero-badge-row">
      <span class="ribbon"><i class="fa-solid fa-fire"></i> 12 Years of Tradition</span>
      <span class="stamp"><i class="fa-solid fa-calendar-day"></i> Fri. Nov 18</span>
    </div>
    ```
  </Step>
</Steps>

<Note>
  The hero section includes a background fire video hosted on Cloudinary (line 117-118). You can replace this with your own video or remove it by deleting the `<video>` element.
</Note>

```html index.html (lines 178-224)
<article class="event-card" data-color="white">
  <div class="event-dot"></div>
  <div class="event-content">

      <span class="time-badge">
        <i class="fa-regular fa-clock"></i> 10:00
      </span>

        <i class="fa-solid fa-microphone"></i> Opening
      </span>
    </div>
    <h3>Event Title</h3>
    <p>Event description goes here.</p>
  </div>
</article>
```

### Event Color Codes

The `data-color` attribute controls the dot color:

<Tabs>
  <Tab title="Available Colors">
    - `data-color="white"` - White dot
    - `data-color="orange"` - Orange/fire color
    - `data-color="gold"` - Gold color
    - `data-color="red"` - Red color
  </Tab>
  
  <Tab title="Custom Colors">
    Add new colors in `styles.css`:
    
    ```css styles.css
    .event-card[data-color="purple"] .event-dot {
      background: linear-gradient(135deg, #8B008B, #DA70D6);
      box-shadow: 0 0 20px rgba(139, 0, 139, 0.5);
    }
    ```
  </Tab>
</Tabs>

## Gallery

Update gallery images with your festival photos:

```html index.html (lines 444-508)
<div class="gallery-masonry" id="galleryGrid">
  <figure class="shot" data-caption="Your photo caption">
    <div class="paper-frame" style="--rotation: -1.5deg;">
      <img src="YOUR_IMAGE_URL" 
           alt="Photo description" 
           loading="lazy" />
    </div>
    <div class="shot-overlay">
      <span class="shot-caption">Your photo caption</span>
    </div>
  </figure>
  <!-- Add more photos -->
</div>
```

### Photo Frame Rotation

Each frame can have a slight rotation for a scrapbook effect:

```html
style="--rotation: -1.5deg;"  <!-- Rotate left -->
style="--rotation: 1.2deg;"   <!-- Rotate right -->
style="--rotation: 0deg;"     <!-- No rotation -->
```

<Tip>
  Use rotation values between -2deg and 2deg for a natural, subtle effect.
</Tip>

## Navigation Menu

Update the main navigation links:

```html index.html (lines 67-75)
<div class="nav-links" id="navLinks">
  <a href="#hero">Home</a>
  <a href="#festival">The Festival</a>
  <a href="#programa">Program</a>
  <a href="#vivo">Live Stream</a>
  <a href="#galeria">Gallery</a>
  <a href="#contacto">Contact</a>
  <a href="#vivo" class="live-cta">
    <i class="fa-solid fa-tower-broadcast"></i> LIVE
  </a>
</div>
```

<Warning>
  **Anchor Links**: The `href` values must match the `id` attributes of your sections. For example, `href="#festival"` links to `<section id="festival">`.
</Warning>

## Content Checklist

Before launching, ensure you've updated:

- [ ] Festival name and logo (hero section)
- [ ] Subtitle and location
- [ ] Festival description and story
- [ ] Event timeline (all events, times, descriptions)
- [ ] Gallery photos (at least 6-8 images)
- [ ] Sponsor logos and names
- [ ] Social media links
- [ ] Footer information
- [ ] Meta tags (title, description, Open Graph)
- [ ] Navigation menu items
- [ ] Live stream URL (in `script.js`)
- [ ] Festival date (in `script.js`)

<Tip>
  **Testing**: After making content changes, test all links and ensure images load correctly. Use browser DevTools to check for any 404 errors.
</Tip>
