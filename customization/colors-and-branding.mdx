## Overview

The Mikunki Shapshico website uses a comprehensive system of CSS variables that make it easy to customize colors, spacing, typography, and visual effects throughout the entire site.

## Color Palette

All colors are defined as CSS variables in `styles.css` starting at line 7. You can customize these to match your festival's brand identity.

## Applying Custom Colors

<Steps>
  <Step title="Open styles.css">
    Locate the `:root` selector at the top of `styles.css` (line 7)
  </Step>
  
  <Step title="Modify color variables">
    Update the hex values for your chosen colors:
    
    ```css
    :root {
      /* Your custom colors */
      --red-main: #8B0000;      /* Darker red */
      --fire-1: #FF4500;         /* Orange-red */
      --gold: #FFA500;           /* Orange gold */
    }
    ```
  </Step>
  
  <Step title="Test your changes">
    Open `index.html` in your browser to see the updated color scheme
  </Step>
</Steps>

## Typography

The site uses two Google Fonts that complement the festival aesthetic:

<ParamField path="--heading" type="font-family" default="'Bebas Neue', sans-serif">
  Display font for headings and titles - bold, condensed, impactful
</ParamField>

<ParamField path="--body" type="font-family" default="'Poppins', sans-serif">
  Body text font - clean, modern, highly readable
</ParamField>

### Changing Fonts

<Accordion title="How to use different Google Fonts">
  1. Visit [Google Fonts](https://fonts.google.com) and select your fonts
  2. Update the font link in `index.html` (line 34):
  
  ```html
  <link href="https://fonts.googleapis.com/css2?family=Your+Font:wght@300;400;600;700&display=swap" rel="stylesheet" />
  ```
  
  3. Update the CSS variables in `styles.css`:
  
  ```css
  :root {
    --heading: "Your Display Font", sans-serif;
    --body: "Your Body Font", sans-serif;
  }
  ```
</Accordion>

## Visual Effects

The site includes several CSS variable-based visual effects:

### Glow Effects

```css styles.css
:root {
  /* Red glow for fire elements */
  --glow-red: 0 0 40px #FF4400, 0 0 80px rgba(204, 0, 0, 0.5);
  
  /* Gold glow for highlights */
  --glow-gold: 0 0 30px rgba(255, 215, 0, 0.4);
}
```

### Shadows

```css styles.css
:root {
  /* Card shadow */
  --shadow-card: 0 14px 28px rgba(0, 0, 0, 0.28);
  
  /* Elevated element shadow */
  --shadow-elevated: 0 20px 40px rgba(0, 0, 0, 0.35);
}
```

### Transitions

```css styles.css
:root {
  --transition-fast: 0.2s ease;
  --transition-base: 0.3s ease;
  --transition-slow: 0.5s ease;
}
```

## Testing Your Changes

After customizing colors and branding:

1. **Visual inspection**: Open `index.html` in multiple browsers
2. **Mobile testing**: Check responsive behavior on different screen sizes
3. **Dark sections**: Verify colors work well on both light and dark backgrounds
4. **Hover states**: Test interactive elements (buttons, links, cards)
5. **Accessibility**: Use browser DevTools to check contrast ratios

<Tip>
  **Quick preview**: Use browser DevTools to test color changes live before editing the CSS file. Right-click any element, select "Inspect", and modify CSS variables in the Styles panel.
</Tip>
